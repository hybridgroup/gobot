LED

Driver

Allows user to interact with LEDs through digital output and PWM (Pulse Width Modulation)

How to establish a connection and setup the driver

JavaScript

var Cylon = require('cylon');

Cylon.robot({
  connection: {name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0'},

  device: {name: 'led', driver: 'led', pin: 13},
  ...

CoffeeScript

Cylon = require 'cylon'

Cylon.robot
  connection:
    name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0'
  ...

Commands:

turnOn
Turns LED on.
turnOff
Turns LED off.
toggle
Turns the LED on, or off, depending on if it is already off, or on, respectively.
brightness(brightness_int) → true | nil
Sets brightness of the led to the specified brightness value passed to brightness(brightness_int) using PWM, brightness can be any integer value between 0 and 255.

Events

start
Sent when the device has been started and is ready to use.

How to use

This example turns an LED on and off.

JavaScript

var Cylon = require('cylon');

Cylon.robot({
  connection: {name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0'},

  device: {name: 'led', driver: 'led', pin: 13},

  work: function(my) {
    every((1).second(), function(my) {
      my.led.toggle();
    });
  }
}).start();

CoffeeScript

Cylon = require 'cylon'

Cylon.robot
  connection:
    name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0'

  device:
    name: 'led', driver: 'led', pin: 13

  work: (my) ->
    every 1.second(), -> my.led.toggle()

.start()
