Roomba

Repository Issues

The Roomba is a simple robot that usually cleans floors.

For more info about the Roomba platform click here.

How to install

Installing Cylon.js with roomba support is pretty easy using ruby gems. 

gem install Cylon.js-roomba

How to use

This small program controls a Roomba. 

require 'Cylon.js'

connection :roomba, :adaptor => :roomba, :port => '/dev/ttyUSB0'
device :roomba, :driver => :roomba, :connection => :roomba

work do
  roomba.safe_mode
  roomba.nudge_left
  roomba.nudge_right
  roomba.nudge_right
  roomba.nudge_left
end
