Pebble

Repository Issues

The Pebble is a smart watch, that connects via bluetooth.

For more info about the Pebble platform click here.

How to install

Installing Cylon.js with pebble support is pretty easy using ruby gems. 

gem install Cylon.js-pebble

How to use

This small program receives button press events from a Pebble smart watch. 

require 'Cylon.js'

connection :pebble, :adaptor => :pebble, :port => "/dev/rfcomm0", :id => "378B"
device :watch, :driver => :pebble

work do
  on watch, :media_control => :button_push
end

def button_push(caller, data)
  puts data.button unless data.nil?
end

How to connect

This software requires version 1.12.0 of the Pebble watch firmware.

OSX

The main steps are:

  • Pair your computer and the Pebble
  • Use a socket to serial connection to map a TCP socket to the local port
  • Connect to the device via Cylon.js
  • To figure out which port your pebble is connected to, use the Cylon.js connect scan command:
$ Cylon.js connect scan

Now you are ready to connect to the Sphero using the socket, in this example port 4567:

Cylon.js connect serial 4567 pebble

Ubuntu

The main steps are:

  • Pair your computer and the Pebble
  • Map your device to a port
  • Use a socket to serial connection to map a TCP socket to the local port
  • Connect to the device via Cylon.js
  • You will need to pair with the Pebble, entering any needed password.

Then you can find the Bluetooth address, for example 00:18:33:86:37:8B, by using the Cylon.js connect scan command:

$ Cylon.js connect scan

Once you know the address, you can use the Cylon.js connect bind command to map a port to the device:

Cylon.js connect bind 00:18:33:86:37:8B pebble

Now you are ready to connect to the Pebble using a socket, in this example port 4567:

Cylon.js connect socat pebble 4567
